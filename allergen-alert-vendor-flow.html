Allergen Alert — Vendor Communication Flow

How a diner's allergen alert travels from their profile to the kitchen — and where the team needs to step in.

Overview
The long-term vision is for the Chef Dashboard to be every vendor's primary tool — where orders, allergen alerts, and confirmations all live in one place. We're not there yet.
Right now, most vendors still operate out of their email inbox, and a hard switch to dashboard-only would break the workflows we depend on to get orders fulfilled. So for v1, we're building backwards: vendors keep working exactly as they do today, and we layer one new accountability step on top for any order that contains allergen alerts. As dashboard adoption grows, the email paths become the fallback — and eventually, the dashboard becomes the gate.

New Step · Final Alert
Final Alert + FP Notification Fire Simultaneously
The final vendor alert is sent via channel preference at the same moment an automated alert fires to the FP/ops team for the market.
  • Final alert timing is anchored to pick-up time — exact window TBD by product.
  • FP alert is modeled on the existing decline alert pattern.
  • If the vendor confirms after receiving the final alert, it is still logged — but FP has already been notified.
HUNGRY Team — Intervention
FP Team Follows Up Directly
FP team contacts the vendor to confirm allergen alerts have been reviewed before the order is prepared. Resolution is recorded per FP process.
Vendor · Kitchen
Kitchen Prepares the Order
Allergen alerts are visible across every surface:
  • 📋Chef Dashboard — allergen column in the group order table + summary banner above
  • 🏷️Printed label — "ALLERGEN:" printed beside the diner's name on every bag
  • 📄PDF order summary — banner at the top + alert shown inline under each item
  • 📑Order manifest — allergen alert shown as a line within the diner's order
HUNGRY Team — If an Incident Is Reported
Full Acknowledgment Log Available for Every Order
If an allergen-related issue comes up after the order is fulfilled, the team has a complete paper trail to reference.
  • 🔍Admin view — acknowledgment log accessible per order
  • 📊Chef Network — confirmation status visible as a column in the order status table
  • 🕐Every log entry captures: timestamp · order ID · path used
